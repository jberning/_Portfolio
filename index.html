Mission

Connecting the Computer to the Human

Every Computer program needs a little help to connect to the end user. That's where I come in; creating the interface to bridge the communtiation between them.

Philosopy

Communication is a basic need for any entity. Communication is achieved by inputs, receiving of information, and outputs, sending information. We are all familiar with human communication through the five senses that offer a richness and versatitily that is unique to our species. In web technologies, a computer is limited to two inputs; keyboard and mouse, and two outputs; visuals and sound. Additionally, modern web has deprecated use of sound for a more pleasing experience.

Why it matters

So a web application is left with only visual cues to comunicate with its users. For example, when it receives input of a mouse pointer hovering over a button, it can only output a visual cue that this button element will perform an action.

It is then my job to facilitate the communication between the computer to the human
